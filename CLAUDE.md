## Project Overview

This is a health data analysis project for INFOST 582 that combines and analyzes personal health data from Apple Health and Garmin exports.

## Project Structure

- **data/raw/**: Original health data exports (may be gitignored for size/privacy)
- **data/cleaned/**: Processed data ready for analysis
- **scripts/cleaning/**: Data cleaning scripts
- **scripts/merging/**: Scripts to combine Apple Health + Garmin data
- **scripts/analysis/**: Analysis code
- **sql/**: Database queries for database assignment
- **docs/**: Assignment proposals and exported visualizations
